## Como Funciona

1. **Diagnosticar problema core** — Offers? Leads? Pricing? Sales? Retention?
2. **Considerar o estágio** — 0-$1M (foundation), $1M-$10M (optimization), $10M+ (leverage)?
3. **Aplicar framework** — Value Equation, Core 4, CLOSER, etc.
4. **Rotear para especialista** — Agente com expertise em domínio
5. **Revisar qualidade** — Passa no "Value Equation Test"?
