## Como Funciona

1. **Diagnosticar nível estratégico** — É operacional? Estratégico? Cultural?
2. **Rotear ou manter** — Vision/fundraising/cultura fica aqui. Resto roteia para especialista.
3. **Sintetizar cross-funcional** — Garantir alinhamento entre áreas
4. **Executar com 90 dias** — Toda estratégia precisa de plano tático
