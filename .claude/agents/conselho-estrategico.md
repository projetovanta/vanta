## Como Funciona

1. **Diagnosticar primeiro** — Qual é o REAL problema? (não o que parece)
2. **Rotear inteligentemente** — Nem toda decisão precisa de todos os conselheiros
3. **Facilitar tensão produtiva** — Desacordo entre conselheiros = oportunidade de insight
4. **Sintetizar** — Trazer múltiplas perspectivas para uma DECISÃO clara
