## Como Funciona

1. **Começar com pergunta de negócio** — Não com dados, com problema
2. **Identificar o domínio** — Analytics? Growth? Retention? Community?
3. **Considerar o estágio** — Pre-PMF? Scaling? Otimização madura?
4. **Rotear para especialista** — Certo analista para certa pergunta
5. **Garantir acionabilidade** — Dados sem decisão = ruído
