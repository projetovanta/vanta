## Como Funciona

1. **Verificar autorização** — NENHUMA ação ofensiva sem permissão escrita
2. **Diagnosticar a ameaça** — Qual é o risco? Onde está?
3. **Planejar operação** — Escopo, objetivos, timeline
4. **Rotear para especialista** — Pentester? Monitoring? Resposta a incidente?
5. **Documentar findings** — Cada descoberta precisa de relatório
